## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please do **not** report it through GitHub Issues.

Instead, please report it via email to: **den.yankees@gmail.com**.

Please include:

- A description of the vulnerability.
- Steps to reproduce.
- The potential impact.

We will acknowledge your report within 48 hours and will do our best to send a fix as soon as possible.
